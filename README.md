# SpellRead

Мобильное приложение для ведения читательского дневника.

## Описание

Приложение позволяет пользователю:

- отслеживать прогресс чтения книг
- добавлять заметки и цитаты
- анализировать статистику чтения
- получать напоминания

## Цель проекта

Разработка удобного инструмента для систематизации процесса чтения и повышения читательской активности пользователя.

## Стек технологий

- Kotlin
- Jetpack Compose
- Navigation Compose
- Material 3
- Room 
- Hilt 

## Архитектура

Проект построен с использованием многослойной архитектуры:

- `ui` — пользовательский интерфейс
- `domain` — бизнес-логика
- `data` — работа с данными
- `navigation` — маршрутизация
- `di` — внедрение зависимостей
